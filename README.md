# BLEVE Overpressure Prediction using Machine Learning

## 📌 Overview

This project develops a machine learning regression model to predict **peak overpressure generated by Boiling Liquid Expanding Vapour Explosions (BLEVEs)**.

BLEVEs are high-risk industrial events involving pressurised vessels containing liquefied gases. Predicting the resulting blast pressure is important for safety analysis, hazard assessment, and infrastructure protection.

The project implements a complete machine learning workflow including data preprocessing, physics-informed feature engineering, model comparison, hyperparameter tuning, and ensemble learning.

---

## 🎯 Objective

Predict peak explosion pressure at different sensor locations using BLEVE scenario parameters such as:

- Tank properties
- Explosion conditions
- Thermodynamic characteristics
- Sensor position
- Obstacle geometry

---

## 🧹 Data Processing

The preprocessing pipeline includes:

- Missing value handling using median imputation
- Duplicate removal
- Categorical feature cleaning and encoding
- Outlier detection and filtering
- Feature scaling
- Target log transformation to handle skewed pressure distributions

---

## ⚙️ Feature Engineering

Created 23 physics-inspired features to improve prediction accuracy, including:

- Tank volume features
- Pressure-volume relationships
- Explosion energy estimations
- Temperature-based features
- Sensor distance calculations
- Spatial positioning features

These engineered variables helped capture non-linear blast behaviour more effectively.

---

## 🤖 Models Implemented

Several regression models were developed and evaluated:

| Model | Purpose |
|---|---|
| ElasticNet | Linear baseline comparison |
| Support Vector Regression | Kernel-based non-linear modelling |
| XGBoost Regressor | Gradient boosting model |
| Neural Network | Deep learning regression |

---

## 🏆 Final Approach

The final solution uses a weighted ensemble combining:

### XGBoost
- Captures complex feature interactions
- Performs strongly on structured/tabular data

### Neural Network
- Learns non-linear representations
- Provides complementary predictions

Combining both models improved overall prediction stability and accuracy.

---

## 📊 Results

| Metric | Score |
|---|---|
| Validation MAPE | 0.1125 |
| Test MAPE | 0.17849 |
| Validation R² | 0.9720 |

---

## 🔍 Key Insights

Feature importance analysis showed:

- Sensor location was one of the strongest predictors
- Spatial positioning significantly influenced pressure values
- Ensemble learning improved generalisation compared with individual models
